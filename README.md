# openFundReminder [W.I.P.]

## 📖介绍
每天晚上才能看基金收益也太慢了，整个下午3点结束就发送预估收益的webapp多爽啊。预计还会写一个2点45时给一个预估收益，方便调仓。

## 📦安装
```bash
git clone
cd openFundReminder
```

## 🚀使用
```bash
python main.py
```
